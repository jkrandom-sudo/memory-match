# 🧠 记忆翻牌 (Memory Match Game)

一个用 Python 开发的**控制台版记忆翻牌游戏**，支持丰富的交互功能和界面。

## 截图预览 (Console)

```

  Moves: 3  |  Matched: 2/16  |  Time: 00:15

      A   B   C   D 
    ┌┬───┬───┬───┬───┐
   1 │ 🍞 │ 🍓 │ ❓ │ ❓ │
    ├┼───┼───┼───┼───┤
   2 │ ❓ │ ❓ │ ❓ │ ❓ │
    ├┼───┼───┼───┼───┤
   3 │ ❓ │ 🍞 │ ❓ │ ❓ │
    ├┼───┼───┼───┼───┤
   4 │ ❓ │ ❓ │ 🍓 │ ❓ │
    └┴───┴───┴───┴───┘
```

## 功能特性

| 功能 | 说明 |
|------|------|
| 🎮 **核心玩法** | 翻开两张卡片，匹配相同图案的卡片对 |
| 📏 **多级难度** | Easy (4×4), Medium (4×6), Hard (6×6) |
| 🌏 **双语支持** | 简体中文 / English，游戏内实时切换 |
| 🔊 **音效系统** | 启用/关闭音效，音量 0-100 可调 |
| 🏆 **成绩记录** | 分数自动保存，支持历史最高分排行 |
| ⏱ **实时计时** | 游戏内显示已用时 (MM:SS) |
| 🎨 **精美界面** | 使用 Unicode 边框和 emoji 装饰卡片 |
| ⏸ **暂停功能** | 游戏过程中随时暂停/继续 |
| ❓ **帮助系统** | 内置 How to Play 指南 |

## 快速开始

### 环境要求
- Python 3.8+
- 无第三方依赖（仅使用标准库）

### 运行游戏

```bash
cd memory-match
python3 game.py
```

### 运行测试

```bash
python3 game_test.py
```

## 操作指南

### 主菜单
| 按键 | 功能 |
|------|------|
| `1` | 开始新游戏 |
| `2` | 选择难度 (Easy/Medium/Hard) |
| `3` | 查看最高分排行 |
| `4` | 设置 (语言/音效/音量) |
| `5` | 游戏说明 |
| `6` | 退出游戏 |

### 游戏进行中
| 按键 | 功能 |
|------|------|
| `a1`~`d4` | 输入卡片坐标翻开 |
| `p` / `pause` | 暂停/继续 |
| `h` / `help` | 显示帮助 |
| `s` | 显示当前统计 |
| `r` | 重新开始 |
| `q` / `quit` | 返回主菜单 |

### 设置
| 选项 | 功能 |
|------|------|
| 语言 (Language) | 简体中文 / English 切换 |
| 音效 (Sound) | ON / OFF |
| 音量 (Volume) | 0~100 调节 |

## 游戏规则

1. 所有卡片初始为背面朝上（❓）
2. 每次翻开两张卡片
3. 若图案相同则配对成功，保持翻开状态
4. 若图案不同则翻回背面
5. 所有卡片配对完成即为胜利
6. 分数 = max(1000 - 步数×10 - 秒数×2, 0)

## 分数系统

- 每次游戏结束自动保存分数
- 高分榜显示 Top 10 记录
- 包含玩家名、分数、难度、步数、用时、日期
- 最高分带有 ⭐ 标记

## 技术栈

- **语言**: Python 3 (标准库)
- **界面**: 控制台字符界面, Unicode 边框+emoji
- **持久化**: JSON 文件 (scores.json, settings.json)
- **测试**: unittest, subprocess 冒烟测试
